# 🛝 The Cosmic Playground

> *Everything in astrology explained as kids on a playground — simple, fun, and surprisingly accurate.*

A single-file interactive astrology reference and AI-powered natal chart reader, built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies, no install.

---

## ✨ What It Is

The Cosmic Playground uses one central metaphor throughout: **astrology as a magical playground.** Every planet, sign, house, asteroid, aspect, and pattern is explained as a kid, a zone, or a dynamic between kids — making abstract astrological concepts immediately intuitive and memorable.

It also includes a **live AI-powered natal chart calculator** that generates a full personalized reading in the Playground voice using the Anthropic API.

---

## 📖 What's Inside

### ⭐ Zodiac Signs
All 12 signs with personality breakdowns, playground analogies, and element/modality tags.

### 🏠 The 12 Houses
Each house as a different zone of the playground — from the spotlight stage (House 1) to the shadow zone (House 12).

### 🪐 Planets
The 10 main planets as the loudest, most unmissable kids on the playground — each with their role, vibe, and what they govern.

### 🌸 Asteroids & Minor Bodies
Lilith, Chiron, Ceres, Pallas, Juno, Vesta, Eris, and Sedna — the quieter, more layered supporting cast.

### 🔭 Special Chart Points
Ascendant, Descendant, MC, IC, Nodes, Part of Fortune, Vertex, Lot of Spirit, and Black Moon Lilith — the invisible rules and forces that shape the whole playground.

### 🔗 Aspects
How planets talk to each other — Conjunction, Opposition, Trine, Square, Sextile, Quincunx, Semi-Sextile, and Grand Trine explained as kid interactions.

### ↩️ Retrogrades
All 8 retrogradable planets with frequency, duration, what the energy means, and what to do (or avoid) during each. Includes a note on natal retrograde placements.

### 🔥 Elements & Modalities
The two secret filing systems behind the 12 signs — Fire/Earth/Air/Water and Cardinal/Fixed/Mutable — with full playground analogies for each.

### 🌀 Chart Patterns
The group dynamics: Stellium, Grand Trine, T-Square, Grand Cross, Grand Sextile, Yod, Bucket/Splay, and Bundle — when planets form shapes that shape a life.

### 🔮 Your Chart *(AI-powered)*
Enter your birth date, time, place, and name — and receive a full personalized natal chart reading in Cosmic Playground style, generated live by Claude AI. Includes your Big Three, all planetary placements, dominant element and modality, key themes, and a narrative summary written just for you.

> **Note:** The chart calculator requires the app to be hosted on a live URL (GitHub Pages, Netlify, etc.) to make API calls. It will not work when opened as a local file.

## 🚀 How to Use

### For the reference sections (no hosting needed)
```
1. Download playground-cosmos-app.html
2. Open it in any browser
3. Use the tab nav to explore each section
```

### For the natal chart calculator (hosting required)

The easiest way is GitHub Pages — free, fast, and takes about 2 minutes:

```
1. Push playground-cosmos-app.html to your GitHub repo
2. Go to your repo → Settings → Pages
3. Under "Source" select: Deploy from a branch
4. Choose: main branch → / (root) → Save
5. Visit https://yourusername.github.io/your-repo-name/
6. Click "Your Chart", enter your birth details, cast your chart ✨
```

Alternatively, drag the file into [Netlify Drop](https://app.netlify.com/drop) for an instant live URL.

## 🎨 Design

- **Fonts:** Playfair Display (headings) + DM Sans (body) via Google Fonts
- **Palette:** Deep space dark (`#060412`) with gold, rose, teal, and lavender accents
- **Animations:** Twinkling star field, floating hero icon, hover card lifts
- **Responsive:** Mobile-friendly with single-column grid breakpoints
- **Zero dependencies:** Pure HTML/CSS/JS — no build step, no node_modules
- **AI layer:** Anthropic API (Claude Sonnet) for the natal chart reading
